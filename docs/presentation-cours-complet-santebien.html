Ce qu'on peut dire au professeur

Notre projet SanteBien applique les concepts du cours sur un mini-modèle IA non diagnostique. Nous avons mesuré la baseline FP32, puis appliqué une quantification INT8. Le modèle passe de 600 bytes à 154 bytes, soit -74,33 % de taille. L'accuracy reste à 100 % sur notre échantillon et l'accord FP32/INT8 est de 100 %, donc nous n'observons pas de perte de qualité. La latence La latence INT8 augmente de 29,74 % dans cette implémentation Python, mais reste sous le seuil. Ces résultats sont protégés par une Green CI : si compression, accuracy, accord, taille, latence ou CO₂ régressent, le pipeline échoue.
